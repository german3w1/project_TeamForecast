Detailed Description

App settings modal dialog.

This class implements app settings dialog that allows:
Set default app theme modes:
•Light. App sets light theme when starts
•Dark. App sets dark theme when starts
•Current. App always "remember" theme switched before it closes and sets that theme when starts
Locations save mode:
•None. App saves no locations
•All. Saves all opened locations when app closes
•Only favorite. When app closes it saves only locations which marked as favorite
Forecasts update interval:
•From 0(disabled) to 24 hours as maximum interval
