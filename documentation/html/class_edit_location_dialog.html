Detailed Description

Modal dialog used for editing data of the location widget from which it was called.

If ok_btn is clicked then edits this location widget with entered in this dialog label, latitude and longitude.
If need_update was switched to true (latitude or longitude were changed) after this dialog is opened, then location widget with edited data updates after dialog closes
