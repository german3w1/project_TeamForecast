Detailed Description

Base widget for Current-, Hourly- and Daily- weather forecast widgets.

Provides an opportunity to expand/collapse content placed in hidden_layout for all inheriting it classes.
Also contains some helper functions like getWindDirection(const int &degrees) and getWeatherIcon(const int &weather_id)
